# Project Overview

## 项目定位

- 项目名称：我们的空间
- 形态：微信小程序 + 微信云开发
- 用户范围：仅限两位成员使用
- 产品主题：以“记录”为核心，围绕情侣回忆、纪念日、心愿和空间管理展开

## 当前产品优先级

1. 回忆记录
2. 纪念日
3. 心愿单
4. 空间管理
5. 宠物岛

说明：宠物岛是辅助情绪化模块，不应压过“记录”主线。

## 当前页面入口

### 底部 Tab

- `pages/home/index`：首页
- `pages/wall/index`：时光墙
- `pages/anniversary/index`：纪念日
- `pages/pet/index`：宠物岛
- `pages/wishes/index`：心愿单

### 非 Tab 页面

- `pages/auth/index`：登录授权
- `pages/my/index`：空间设置
- `pages/release/index`：发布回忆
- `pages/moment/detail`：回忆详情
- `pages/diary/index`：日历日记

## 当前已实现能力摘要

- 白名单登录与鉴权
- 首页数据概览与最近回忆展示
- 时光墙发布、图片上传、点赞、评论、地点、多图浏览
- 纪念日管理、里程碑进度、提醒设置
- 心愿单管理与完成状态切换
- 宠物岛基础互动和成长展示
- 日历日记按月和按天查看
- 空间设置与清空数据

## 当前未完成能力摘要

- 纪念日真实订阅消息推送
- 云端定时触发提醒
- 宠物岛深度玩法扩展

## 近期视觉方向

- 页面整体使用暖色、梦幻、柔和的情绪化视觉
- 装饰重点放在页面两侧和底部，避免遮挡中间主要卡片内容
- 首页、纪念日、心愿单已接入这套视觉语言，后续还要继续统一剩余页面
