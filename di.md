# 我们的空间开发文档

## 1. 项目定位

- 项目名称：我们的空间
- 项目形态：微信小程序 + 云开发
- 使用对象：仅限两位成员的私密双人空间
- 当前目标：围绕回忆记录、纪念日、心愿、宠物互动和空间管理持续做二开

## 2. 当前页面与入口

### 2.1 底部 Tab

- 首页
- 时光墙
- 纪念日
- 宠物岛
- 心愿单

### 2.2 非 Tab 页面

- 登录授权页：`pages/auth/index`
- 空间设置页：`pages/my/index`
- 发布回忆页：`pages/release/index`
- 回忆详情页：`pages/moment/detail`
- 日历日记页：`pages/diary/index`

说明：旧的“纪念与心愿合并页”已从路由中移除，纪念日和心愿能力以独立页面为准。

## 3. 数据与云开发

### 3.1 已接入集合

| 集合名 | 用途 |
| --- | --- |
| `users` | 双人白名单、成员身份信息 |
| `space` | 空间基础信息、在一起日期、提醒设置、宠物状态 |
| `moments` | 时光墙内容、图片、点赞、评论、地点信息 |
| `anniversaries` | 自定义纪念日 |
| `wishes` | 心愿清单 |

### 3.2 已接入云能力

- `auth` 云函数：登录白名单校验、空间初始化
- `relationship` 云函数：空间内容的增删改查
- 图片上传：已用于时光墙发布

## 5. 当前功能拆分

### 5.1 首页

- 展示在一起天数、回忆数和最近心情
- 展示最近回忆和地点信息
- 支持直接进入发布回忆

### 5.2 时光墙

- 瀑布式回忆内容浏览
- 回忆发布、编辑展示优化、多图预览
- 点赞、评论、地点和详情查看
- 提供进入日历日记的入口

### 5.3 纪念日

- 在一起日期维护
- 自定义纪念日管理
- 100 天里程碑与年度进度展示
- 应用内提醒提示和提前天数切换

### 5.4 心愿单

- 心愿录入
- 完成状态切换
- 已完成 / 待完成展示
- 页面统计信息展示

### 5.5 宠物岛

- 双宠物状态展示
- 投喂、贴贴、等级成长
- 基础皮肤切换

### 5.6 日历日记

- 月视图查看记录密度
- 查看某一天的已发布内容
- 从指定日期发起记录

### 5.7 空间设置

- 空间资料维护
- 成员昵称维护
- 一键清空空间数据

## 7. 当前验收重点

1. 白名单登录后是否能正确进入空间。
2. 在一起日期是否能正确保存到 `space.startDate`。
3. 时光墙发布、点赞、评论、地点、多图预览是否正常。
4. 纪念日进度条、提醒开关、提前天数设置是否正常。
5. 心愿单新增、完成切换、删除是否正常。
6. 宠物岛投喂、互动、等级和皮肤切换是否正常。
7. 日历日记按月切换、按天查看、指定日期记录是否正常。
