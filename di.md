# 我们的空间开发文档

## 1. 项目简介

- 项目名称：我们的空间
- 项目类型：微信小程序 + 云开发
- 使用对象：仅限两位成员
- 核心目标：记录回忆、管理纪念日、维护心愿、逐步扩展宠物互动

## 2. 当前项目状态

### 2.1 已完成

| 模块 | 已实现内容 |
| --- | --- |
| 登录鉴权 | 基于 OpenID 的两人白名单登录 |
| 云开发 | 已接入 `users`、`space`、`moments`、`anniversaries`、`wishes` |
| 首页 | 展示在一起天数、最近回忆、宠物岛占位信息 |
| 时光墙 | 发布回忆、上传图片、查看详情、删除本人回忆 |
| 纪念日 | 设置在一起日期、添加纪念日、倒计时/正计时、删除纪念日 |
| 心愿单 | 新增心愿、完成切换、删除心愿 |
| 宠物岛 | 已加入底部导航，当前为基础占位页 |
| 空间设置 | 可修改空间名称、双方昵称、空间介绍 |
| 页面状态 | 已补充 loading、空状态、失败提示 |

### 2.2 当前页面结构

| 页面 | 作用 |
| --- | --- |
| 首页 | 展示核心信息和最近回忆 |
| 时光墙 | 查看全部回忆，进入发布页 |
| 纪念日 | 管理在一起日期和纪念日 |
| 宠物岛 | 当前为占位页，后续接互动功能 |
| 心愿单 | 管理想一起完成的愿望 |

### 2.3 当前底部导航

`首页 / 时光墙 / 纪念日 / 宠物岛 / 心愿单`

## 3. 数据结构

| 集合名 | 当前用途 |
| --- | --- |
| `users` | 两位成员白名单与身份信息 |
| `space` | 空间基础配置，如名称、双方昵称、介绍、在一起日期 |
| `moments` | 时光墙回忆内容 |
| `anniversaries` | 纪念日数据 |
| `wishes` | 心愿单数据 |

## 4. 关键功能说明

### 4.1 首页

- 展示在一起天数、回忆数、最近心情。
- 支持直接设置在一起日期。
- 展示最近 3 条回忆。
- 展示宠物岛占位信息。

### 4.2 时光墙

- 支持图片上传。
- 支持文字、日期、心情记录。
- 支持查看详情。
- 支持删除自己发布的回忆。

### 4.3 纪念日

- 支持设置在一起日期。
- 支持新增和删除纪念日。
- 支持显示距离纪念日还有多少天或已经过去多少天。

### 4.4 心愿单

- 支持新增心愿。
- 支持标记完成。
- 支持删除。

### 4.5 宠物岛

- 当前仅完成页面入口和基础占位。
- 后续可继续扩展喂养、亲密度、等级、互动。

## 7. 建议开发顺序

1. 先补数据权限规则和基础校验。
2. 再完善时光墙互动功能。
3. 接着补纪念日增强能力。
4. 然后正式开发宠物岛。
5. 最后补日记和提醒功能。
